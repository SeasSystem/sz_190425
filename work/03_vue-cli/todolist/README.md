# vue-cli(2.0)
  1. 安装 : npm i vue-cli -g
  2. 查看版本: vue -V
  3. 查看脚手架的模板 : vue list
  4. 初始化脚手架: vue init webpack 项目名
  5. 启动项目
        开发: npm start
        生产: npm run build
